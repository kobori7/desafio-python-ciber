# Simulação Educacional de Malware com Python

Este projeto tem fins **educacionais** e demonstra, em ambiente controlado, como dois tipos comuns de malware funcionam:

- Ransomware
- Keylogger

O objetivo é compreender o funcionamento dessas ameaças e estudar formas de defesa e prevenção.

⚠️ Nenhum código malicioso real é utilizado. Todas as simulações são feitas de forma segura.

---

## Objetivos

- Entender o funcionamento de ransomware
- Entender como keyloggers capturam dados
- Estudar técnicas de detecção e defesa
- Documentar o processo de aprendizado
