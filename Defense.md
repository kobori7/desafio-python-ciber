# Estratégias de Defesa Contra Malware

## 1. Antivírus e EDR

Ferramentas modernas detectam comportamento suspeito como:

- criptografia massiva de arquivos
- captura de teclado
- conexões suspeitas

## 2. Firewall

Bloqueia conexões externas não autorizadas.

## 3. Sandboxing

Arquivos suspeitos podem ser executados em ambientes isolados para análise.

## 4. Backups

Backups offline impedem perda total de dados em ataques ransomware.

## 5. Treinamento de usuários

Usuários treinados identificam:

- e-mails de phishing
- links suspeitos
- anexos maliciosos

Isso reduz significativamente o risco de infecção.
